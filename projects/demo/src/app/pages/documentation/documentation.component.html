Documentation

Component Inputs

Input Type Description
[data] Object Table data, any valid JSON object.
[settings] Object Table component configuration object, properties described below

Table Configuration

Property Type Default Description
key Object n/a Object keys Column settings
key.headerText string 'Key' Object keys column header text.
key.width string '40%' Object keys column width, example: '40%', '150px'
value Object n/a Object values Column settings
value.headerText string 'Value' Object values column header text.
value.width string '60%' Object values column width, example: '60%', '200px'
options Object n/a Whether to enable or disable the add/edit/delete options.
options.add boolean false Whether to enable or disable the add option.
options.edit Object n/a Whether to enable or disable the edit option.
options.edit.key boolean false Whether to enable or disable the edit the key.
options.edit.value boolean false Whether to enable or disable the edit the value.
options.delete boolean false Whether to enable or disable the delete option.
sortable boolean false Whether the table is sortable or not.
sortDirection 'asc' | 'desc' undefined Sort table by the object keys column with this direction by default.
Applied only when sortable = true.
expandAll boolean false Expand all the objects/arrays children when loading the data the first time.
loadFromFile boolean false Load table data from a local JSON file.
iconPackage 'basic' | 'font-awesome'
| 'material-design' 		'basic' Name of the icon package used for the styling of the table icons.
icons 'Icons' n/a The icons metadata used for the styling of the table icons.
The icons names are:
  • notSorted: when the table is not sorted yet.
  • sortedAsc: when the table is sorted in ascending order.
  • sortedDesc: when the table is sorted in descending order.
  • expand: when the object/array child is expanded.
  • collapse: when the object/array child is collapsed.
  • simpleChild: when the child is simple(key: value).
  • addChild: to add a child.
  • editChild: to edit a child.
  • confirmEditChild: to confirm editing a child.
  • cancelEditChild: to cancel editing a child.
  • deleteChild: to delete a child.
  • uploadFile: to upload a file.
icons.[iconName].innerText string n/a The icon inner text, example: {{ exampleIconInnerText }}
icons.[iconName].class string n/a The icon class, example: {{ exampleIconClass }}
icons.[iconName].color string n/a The icon color, example: {{ exampleIconColor }}
icons.[iconName].html string n/a The icon html and this propriety will override all other icon proprieties,
example:{{ exampleIconHtml }}
