Interactive Features

With the appropriate settings, users can:

  • Add new properties to objects or items to arrays
  • Edit existing keys and values
  • Delete properties or array items
  • Sort data in ascending or descending order
  • Expand or collapse nested objects and arrays
  • Import JSON data from files

Try these features in the demo above by clicking on the various icons.
